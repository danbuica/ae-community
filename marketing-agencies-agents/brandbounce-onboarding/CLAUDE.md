# BrandBounce Onboarding App

## What this is
A Next.js 15 client onboarding wizard for Brand Bounce marketing agency. Clients fill out a 7-step form with Gemini AI assistance. Data saves to `src/data/submissions/` as JSON files. The `/generate-style-guide` skill reads a submission and generates a markdown style guide.

## Key conventions
- **App Router** — all routes under `src/app/`, server components by default, `"use client"` only where needed
- **State** — Zustand store in `src/hooks/use-form-store.ts` holds all form data
- **AI calls** — all Gemini calls go through `POST /api/ai/suggest`, never directly from client
- **File uploads** — saved to `src/data/submissions/uploads/`
- **Styling** — Tailwind v4 + shadcn/ui. Brand Bounce primary = `#7C3AED` (violet), accent = `#F43F5E` (coral). Use `gradient-brand-text` class for gradient text.
- **Animations** — shared variants in `src/lib/animations.ts`, always use `AnimatePresence` for mount/unmount

## Running locally
```bash
npm run dev
```
App runs at http://localhost:3000. Set `GEMINI_API_KEY` in `.env.local` before testing AI features.

## Submissions directory
```
src/data/submissions/
  {uuid}.json          # Form data
  {uuid}-style-guide.md  # Generated by /generate-style-guide skill
  uploads/             # Uploaded logos and guidelines PDFs
```
